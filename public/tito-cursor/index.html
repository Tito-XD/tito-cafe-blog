安装方法

ZIP：解压后右键 install.inf → 选择「安装」。
EXE：双击 Tito-Cursor.exe 按提示安装。
启用：设置 → 个性化 → 主题 → 鼠标指针样式 → 选择 Tito 方案。

指针颜色变浅 / 发白？

部分电脑在开启独显、HDR 或部分游戏环境下，自定义指针可能出现颜色变淡、泛白。 可使用 SoftCursor 将渲染方式改为 Software cursor（软件渲染），通常可恢复正常颜色。

下载 SoftCursor-1.0.zip
1. 解压 SoftCursor-1.0.zip，运行其中的 SoftCursor 程序。
2. 在界面中选择 Software cursor（软件渲染），立即生效，无需常驻后台。
3. 若要恢复默认硬件渲染，再次运行并选择 Hardware cursor。

工具来源：ToastyX SoftCursor（开源）。源文件位于仓库 src/assets/cursor/SoftCursor-1.0.zip。
